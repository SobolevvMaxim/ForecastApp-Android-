## Stack
- Kotlin
- MVVM
- Retrofit
- Room
- Coroutines
- GSON
- Hilt
## Features
- Load default city (Minsk) weather at first start
- Get city forecast by name
- Tap the city to choose it
- Long tap to delete the city
- Navigate to main screen to see week forecast
- Animation between the fragments
